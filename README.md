## MapReduce-Programs

### PrimeSum  
Calculates the sum of all prime numbers present in the input file through MapReduce program.  
- PrimeSumDriver
- PrimeSumMapper
- PrimeSumReducer

### CountIndia
Calculates how many times “India” keyword appears in the given input file through Map-Reduce program.
- CountIndiaDriver
- CountIndiaMapper
- CountIndiaReducer
